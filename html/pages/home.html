Nullus Platform

DevSecOps 인프라 자동화 플랫폼

검증된 CI/CD 베스트 프랙티스 조합(Golden Path)을 선택하고, 노코드 UI로 Kubernetes 기반 DevSecOps 파이프라인을 즉시 구축하는 오픈소스 플랫폼

핵심 기능

DevSecOps Stack 자동 설치
GitLab, ArgoCD, Prometheus 등 스택을 버튼 한 번으로 Kubernetes에 배포
Golden Path 템플릿
프로덕션 검증된 도구 조합 — GitHub+ArgoCD, GitLab All-in-One 등
CI/CD Pipeline 관리
애플리케이션별 파이프라인 배포·이력 관리. Web/API/Batch 템플릿 제공
버전 호환성 보장
검증된 도구 버전 조합만 제공, 예측 불가능한 호환성 이슈 방지
통합 모니터링
클러스터·파이프라인·앱 상태를 하나의 대시보드에서 모니터링
Role 기반 권한 관리
Admin / DevOps Engineer / Developer 역할 기반 접근 제어
